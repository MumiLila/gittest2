Tag Attribute คำอธิบาย
h สำหรับกำหนดหัวข้อ
p สำหรับการขึ้นย่อหน้าใหม่
div จะใช้จัดกลุ่มเนื้อหาแบบบล็อก
img src ใช้กำหนดตำแหน่งและชื่อของภาพ
alt เป็นข้อความให้แสดงแทนภาพ
title เป็นข้อความหรือชื่อภาพที่แสดงแบบ tooltip
width ความกว้างของภาพ
height ความสูงของภาพ
a href กำหนดเป้าหมายของการเชื่อมโยง
target กำหนดเป้าหมายที่จะเปิดลิงค์นั้น
id กำหนดชื่ออ้างอิงกรณีที่จะเชื่อมโยงภายในเพจ
ul type=disc เป็นวงกลม
type=circle เป็นวงกลมโปร่งแสง
type=square เป็นสี่เหลี่ยม
ol type=1 เป็นตัวเลข
type=a เป็นอักษรภาษอังกฤษแบบพิมพ์เล็ก
type=A เป็นอักษรภาษอังกฤษแบบพิมพ์ใหญ่
type=i เป็นตัวเลขโรมันแบบพิมพ์เล็ก
type=I เป็นตัวเลขโรมันแบบพิมพ์ใหญ่
table border กำหนดเส้นขอบ
form method กำหนดรูปแบบในการส่งข้อมูล
action กำหนดเพจปลายทางด้านเซิร์ฟเวอร์ที่จะรับข้อมูลจากฟอร์ม
target กำหนดเป้าหมายในการส่งข้อมูลจากฟอร์มออกไป
autocomplete กำหนดให้มีตัวเลือกสำหรับเติมข้อมูลในฟอร์มหรือไม่
novalidation การตรวจสอบข้อมูลบางแอตทริบิวจะถูกยกเลิก
button สร้างปุ่มสั่งงาน
