Tag ความเหมือน ความต่าง
hr&br ใช้สำหรับจัดการข้อความ hr เป็นสร้างเส้นแนวนอน แต่ br เป็นขึ้นบรรทัดใหม่
ul&ol ใช้ในส่วนของการแสดงรายการทั้งคู่ ul สำหรับทำลิสต์รายการแบบบุลเล็ต แต่ ol สำหรับทำลิสต์รายการแบบลำดับตัวเลข
select&checkbox&radio button เป็นอิลิเมนต์เพื่อใช้ในการสื่อสารกันระหว่างuserกับserver select ตัวเลือกแบบ drop-down list ถูกใช้สร้างเพื่อให้ผู้ใช้งานนำไปใช้สร้างตัวเลือกที่มีการเลือกใช้งานได้หลากหลายมากกว่า 1 ตัว
checkbox จะมาในรูปแบบของกล่องสี่เหลี่ยม ด้านในเป็นพื้นที่ว่าง เพื่อให้ผู้ใช้ กดกากบาท หรือ กดเครื่องหมายถูก เพื่อเลือกสิ่งที่เราต้องการหรือเลือกได้มากกว่านั้น โดยไม่จำเป็นว่าจะต้องเลือกแค่อย่างเดียว ส่วน
radiobutton จะใช้ในการเลือกแค่ตัวเลือกนั้นตัวเลือกเดียว ไม่สามารถเลือกได้หลายตัวเลือก จะอยู่ในรูปของวงกลมเป็นพื้นที่ว่างๆ ถ้าเราต้องการเลือก สามารถการกดจุดเข้าไปในวงกลมนั้นได้
div&span เป็นการจัดการกลุ่มเนื้อหาเหมือนกัน div จะใช้จัดกลุ่มเนื้อหาแบบบล็อก โดยจะทำให้อิลิเมนต์ที่อยู่ต่อจาก div จะแยกออกเป็นบรรทัดใหม่ ต่างจาก
span จะจัดกลุ่มอยู่ในบรรทัดของอิลิเมนต์นั้นได้
script&style เป็นtagช่วยให้htmlทำตอบโจทย์การใช้งานได้ดีขึ้น script ภาษาที่เป็น Script ที่อยู่ในเว็บไซต์ (ใช่ร่วมกับ HTML) เพื่อให้เว็บไซต์ดูมีการเคลื่อนไหว สามารถตอบสนองผู้ใช้งานได้มากขึ้น
style ช่วยในการกำหนด CSS เข้าไปใน HTML เพื่อความสวยงาม
