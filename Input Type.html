type ลักษณะ
type="text" กำหนดว่าเป็นอิลิเมนต์ชนิด text
type="password" กำหนดว่าเป็นอิลิเมนต์ชนิด password
type="submit" สร้างปุ่ม submit
type="reset" สร้างปุ่ม reset
type="redio" สร้างรายการตัวเลือกที่เลือกได้ตัวเดียว
type="checkbox" สร้างรายการตัวเลือกที่เลือกได้มากกว่า1ตัว
type="button" สร้างปุ่ม
type="color" สร้างปุ่มเลือกสี
type="date" สร้างปุ่มวันที่
type="datetime and local" สร้างช่องข้อมูลเวลาและวันที่
type="Email" ช่องใส่email
type="file" สร้างช่องเพื่อทำการเลือกupload file
type="month" สร้างปุ่มเลือกเดือน
type="number" สร้างช่องข้อมูลตัวเลข
